## Setup

1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the root directory and set up your database:
   ```bash
   DB_HOST=localhost
   DB_PORT=3306
   DB_USER=root
   DB_PASSWORD=yourpassword
   DB_NAME=yourdb
   ```

4. Set up the MySQL database by creating the required tables and entities.

5. Run the application:
   ```bash
   npm run start:dev
   ```

6. Access Swagger documentation at:
   ```
   http://localhost:3000/api
   ```

## Docker

1. Build and run the app with Docker:
   ```bash
   docker-compose up --build
   ```

2. Access the app at:
   ```
   http://localhost:3000
   ```
